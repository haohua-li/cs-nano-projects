## 先做后学

通过一些小型项目学习知识。

这是 [“先学后教”](https://theory.gmw.cn/2021-10/08/content_35215197.htm) 的环节，也就是先确定好任务，通过任务自学，最后再去系统地听课。 

这里的项目都不应该太大。
